 PDF to Audiobook Converter

A Python GUI application that converts PDF files into spoken audio (MP3).

## Features
- Upload PDF file
- Convert text to speech
- Adjustable speech speed
- Voice selection
- Language support (English, Hindi, Kannada, Telugu)
- Progress bar during conversion

## Technologies Used
- Python
- Tkinter
- PyMuPDF (fitz)
- pyttsx3
- gTTS
- Pillow

## How to Run

1. Install required libraries

pip install pymupdf pyttsx3 gtts pillow

2. Run the application

python app.py
